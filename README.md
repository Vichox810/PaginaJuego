# BichoJuegos

BichoJuegos es una tienda en línea de videojuegos que ofrece los mejores precios en Chile. El sitio web incluye una página principal con secciones de bienvenida, videos promocionales, lista de juegos disponibles y un formulario de contacto.


# Características
- **Videos**: Incluye un iframe de YouTube y un video local.
- **Juegos**: Muestra tarjetas de juegos con imágenes, videos en hover, descripciones y precios. Cada juego tiene un botón para añadir al carrito.
- **Formulario de Contacto**: Validación en tiempo real para nombre, apellido, email, asunto y mensaje. Requiere aceptación de términos.
- **Carrito**: Página simple que confirma la adición de un producto al carrito.


# Tecnologías Utilizadas
- HTML5
- CSS3 (con diseño responsivo)
- JavaScript (para validación del formulario)


# Cómo Ejecutar
1. Clona o descarga los archivos del proyecto.
2. Abre index.html en un navegador web moderno.
3. Para el formulario de contacto, interactúa con los campos y envía el formulario para ver la validación.

# Estructura de Archivos
- index.html: Página principal.
- Pagina_videojuegos.css: Estilos para la página principal.
- contacto.js: Lógica de validación del formulario.
- Carrito.html: Página del carrito.
- Carrito.css: Estilos para la página del carrito.

## Notas
- Los videos e imágenes deben estar en el mismo directorio que los archivos HTML para que se carguen correctamente.
- El formulario de contacto no envía datos reales; solo valida y muestra un mensaje de éxito.
